# Title: 
- Version control hosting service for Project

### Date:
- 2023-02-18

### Status:
- Accepted

### Context and Problem Statement
- Which version control hosting service should use?

### Considered Options:
- Github, Gitlab

### Decision:
- Github

### Reasons for the decision:
- Main reason is that creating new PR in GitHub can be done locally in Pycharm, which is very convenient.
